# Sortyard Docs

Documentation for the Sorting Center Simulation hackathon project.

---

## Documents

| Document | What's inside |
|---|---|
| [PRD](./prd/README.md) | Goals, scope, functional and non-functional requirements, user stories, tech stack, roadmap, risks |
| [Theory](./theory/README.md) | DES fundamentals, sorting center architecture, queueing theory, bottleneck detection, conveyor modeling, PRNG |
| [Contracts](./contracts/README.md) | REST API request/response schemas, WebSocket event payloads, error codes |

---

## Recommended reading order

**New team member starting from scratch:**

1. [PRD](./prd/README.md) — what we're building, why, and what's in scope
2. [Theory](./theory/README.md) — domain knowledge and DES concepts needed to build the engine
3. [Contracts](./contracts/README.md) — backend/frontend interface; read this before writing any API or UI code
